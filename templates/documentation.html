System Overview

This system uses a 3D U-Net architecture for brain tumor segmentation and classification.

Supported Formats

  • NIfTI files (.nii, .nii.gz)
  • DICOM files (.dcm)
  • Standard images (PNG, JPG)

Usage

1. Upload your medical image

2. Wait for processing

3. Review the segmentation and classification results
